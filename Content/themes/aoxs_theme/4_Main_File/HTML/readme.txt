Here are some important notices for PhotArtist HTML&CSS:

1) There are 4 Different Columns & Sidebars  Layout. And all columns have parent container 
<div class="middle"> - here I add additional classes to define which layout should be.

	a) <div class="middle cols3_fix">  
		- for 3 columns layout (Sidebar 1|Content|Sidebar 2) blog.html

	b) <div class="middle cols2_fix"> 
		- for 2 columns layout (Sidebar 1|Content) shortcodes-text.html

	c) <div class="middle cols2_wide"> 
		- for 2 columns layout (Sidebar 1|Content Wide) shortcodes-full-cols2.html

	d) <div class="middle cols3_fix sidebar_right"> 
		- for 3 columns layout when sidbears at the right (Content | Sidebar 2| Sidebar 1) page-sidebar-r.html

	
* Note: Class "sidebar_right" can be added to any of a,b,c variants.

2) Used Supersized plugin:
	- Documentations here http://buildinternet.com/project/supersized/
	- Styles that can be edited can be found here: \css\slideshow\
